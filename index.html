คู่มือเริ่มต้นการใช้งานระบบ

ยินดีต้อนรับเข้าสู่หน้าคู่มือการใช้งาน เว็บไซต์นี้จัดทำขึ้นเพื่อใช้ดูประกอบการทำงานของระบบแบบเป็นขั้นตอน

เริ่มต้นใช้งาน → ดูคู่มือทั้งหมด
💡

หมายเหตุ: คู่มือนี้จะอัปเดตเวอร์ชันล่าสุดเสมอผ่านระบบ Static Hosting ของเรา

⚙️ ความต้องการของระบบ

ก่อนเริ่มใช้งาน กรุณาตรวจสอบให้แน่ใจว่าเครื่องของคุณมีคำสั่ง node เวอร์ชัน 18 ขึ้นไปติดตั้งเรียบร้อยแล้ว

🟢

Node.js 18+

JavaScript runtime สำหรับรันโปรเจกต์

📦

npm / pnpm

Package manager สำหรับติดตั้ง dependencies

💻

Terminal

สำหรับรันคำสั่งเริ่มต้นโปรเจกต์

🚀 คำสั่งเบื้องต้น

เปิด Terminal แล้วรันคำสั่งด้านล่างนี้เพื่อเริ่มต้นโปรเจกต์ใหม่:

terminal 
npm install
npm run dev

เมื่อรันสำเร็จ ระบบจะเปิด Local Server ให้คุณสามารถทดสอบใช้งานได้ทันที
